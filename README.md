# open-c-sharp-templates
Simple open source C sharp scripts for beginners

### Comments
```cs
// This is a comment
Console.WriteLine("Hello World!");
```

### Variables
types:
- int
- string
- double
```cs
type variableName = value;
```

### Output

```cs
Console.WriteLine("Hello World!");
```

### Conditions
```cs
if //(condition) 
{
  // block of code to be executed if the condition is True
}
```

### Simple math
```
int mathEx;
mathEx = 1 + 1
```
